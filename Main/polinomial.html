Penjumlahan Polinomial

Diketahui:

f(t) = 2t4 - 4t3 + 5t

g(t) = -3t4 + t3 - 5t2 + t - 3

Hasil dari penjumlahan f(t) + g(t) adalah:

  • A: -t4 - 4t3 - 5t2 + 6t - 3
  • B: -t4 - 3t3 - 5t2 + 6t - 3
  • C: -t4 + 3t3 - 5t2 - 3
  • D: -t4 + 4t3 - 2t2 + t - 3
  • E: -t4 + 3t3 + 5t2 - t - 3

Pembahasan:

  1. Suku (t4):
    ( 2t4 + (-3t4) = -t4 )
  2. Suku (t3):
    ( -4t3 + t3 = -3t3 )
  3. Suku (t2):
    ( 0 + (-5t2) = -5t2 )
  4. Suku (t):
    ( 5t + t = 6t )
  5. Konstanta:
    ( 0 + (-3) = -3 )

Sehingga hasil penjumlahan adalah: -t4 - 3t3 - 5t2 + 6t - 3

Kunci: B

Derajat Polinomial

Diketahui:

Derajat polinomial dari 12x2 - 3x3 + 5x4 - 5x7 + x + 11 adalah ...

  • 4
  • 3
  • 7
  • 6
  • 8

Pembahasan:

Derajat suatu polinom merupakan pangkat dengan angka tertinggi. Pada persamaan ini, pangkat tertingginya adalah 7.

Kunci: C
